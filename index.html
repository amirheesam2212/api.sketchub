APIs provided by Sketchub

Codehub API
Codehub provides API for Java Codes.

GET
 http://api.sketchub.in/codehub/

RESULT
Result will be Array JSON (Listmap JSON), in which Object JSON (Map JSON) will be listed of following keys.
- id (Unique ID of Code issued by Codehub)
- title (Title of Code)
- code (The main code will be here)
- about (The information related to the particular code.
- category (Category of the code)
- source (Link or Name of source)
- androidx_supported (If it is supported to latest Sketchware or not)
- android_old_supported (If it is supported to Sketchware v3.9.9 or less)

Categories for Java Codes.

GET
 http://api.sketchub.in/codehub/category

RESULT
Result will be Array JSON (Listmap JSON), in which Object JSON (Map JSON) will be listed of following keys.
- name (Unique Category Name)
